]]> By: Author https://www.sordum.org/8366/reicon-v1-9-restore-desktop-icon-layouts/comment-page-8/#comment-31726 Wed, 16 Jan 2019 10:02:19 +0000 https://www.sordum.org/?p=8366#comment-31726 You can use a bat file to do this for example:

Backup:
@echo off
ReIcon.exe /S /File “%appdata%\icons.ini”

Restore:
@echo off
ReIcon.exe /R /File “%appdata%\icons.ini”

Or Use shortcut with parameter
